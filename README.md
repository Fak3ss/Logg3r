# Logg3r v1

Keylogger simple en Python para fines educativos, desarrollado con `pynput`. Captura las teclas y las guarda en un txt.

Instala las dependencias:
   ```
   pip install -r requirements.txt
   ```

Ejecuta con el comando:
   ```
   python logg3r.py
   ```
